# APE04-Clases-Objetos-Metodos--Steven-Lopez

##  Descripción del Proyecto

Este proyecto fue desarrollado como parte de la actividad práctica **APE 04** de la asignatura **Algoritmos y Lógica de Programación** de la **Universidad Técnica de Ambato**.

El objetivo es implementar un sistema básico de control de estudiantes y calificaciones utilizando los principios de la **Programación Orientada a Objetos (POO)**, aplicando:

- Clases
- Objetos
- Métodos
- Encapsulamiento
- Constructores
- Métodos Get y Set

El programa fue desarrollado en dos lenguajes de programación:

- C++
- Java

---

##  Objetivo

Registrar información de estudiantes y calcular automáticamente su promedio y estado académico.

Cada estudiante contiene la siguiente información:

- Cédula
- Nombre
- Apellido
- Nota 1
- Nota 2
- Nota 3
- Promedio
- Estado (Aprobado o Reprobado)

---

##  Funcionalidades del Sistema

- Registro de 5 estudiantes
- Ingreso de 3 notas por estudiante
- Validación de notas entre 0 y 10
- Cálculo automático del promedio
- Determinación del estado académico
- Visualización del listado completo
- Conteo de estudiantes aprobados y reprobados

---

##  Condición de Aprobación

Un estudiante aprueba si su promedio es mayor o igual a: 7.0
